<h3>
  <img src="https://api.iconify.design/lucide:rocket.svg?color=%239333ea" width="24" align="absmiddle"> 
  Iniciando o Sistema (Como usar)
</h3>

A operação é extremamente simples:

1. Acesse a página principal do **Khan Academy**.
2. Clique no favorito onde você salvou o **MoonScripts**.
3. O sistema será iniciado silenciosamente.
4. Navegue e **abra a atividade desejada**. O script fará o resto.

> <img src="https://api.iconify.design/lucide:info.svg?color=%233b82f6" width="18" align="absmiddle"> **Dica do Sistema:** Execute o script logo na tela inicial do Khan Academy. Assim, qualquer atividade que você abrir em seguida já estará sob o efeito da automação.

<h3>
  <img src="https://api.iconify.design/lucide:download-cloud.svg?color=%239333ea" width="24" align="absmiddle"> 
  Guia de Instalação Rápida
</h3>

<p>Escolha sua plataforma abaixo e siga os passos para criar o seu botão de ativação:</p>

#### <img src="https://api.iconify.design/lucide:monitor.svg?color=%2364748b" width="20" align="absmiddle"> Computador (Chrome / Edge / Brave)
1. Copie o código do script no topo da página.
2. Clique com o botão direito na sua **Barra de Favoritos** no navegador.
3. Selecione **Adicionar página...**
4. No campo **Nome**, digite: `MoonScripts`
5. No campo **URL**, cole todo o código JavaScript.
6. Clique em **Salvar**.

#### <img src="https://api.iconify.design/lucide:smartphone.svg?color=%2310b981" width="20" align="absmiddle"> Android (Google Chrome)
1. Abra qualquer site no Chrome.
2. Toque nos **Três pontos (⋮)** no canto superior direito e em **Adicionar aos favoritos**.
3. Acesse a sua lista de **Favoritos**.
4. Segure o dedo sobre o favorito que acabou de criar e toque em **Editar**.
5. No campo **Nome**, coloque `MoonScripts`.
6. No campo **URL**, apague o conteúdo atual e **cole o nosso script**.

#### <img src="https://api.iconify.design/bi:apple.svg?color=%2364748b" width="20" align="absmiddle"> iPhone & iPad (Safari)
1. Abra qualquer site no Safari.
2. Toque no botão de **Compartilhar** (quadrado com seta para cima).
3. Selecione **Adicionar Favorito**.
4. Abra o ícone de **Livros** (Favoritos) na barra inferior.
5. Toque em **Editar** e selecione o favorito criado.
6. Renomeie para `MoonScripts` e cole o nosso script no campo do **URL**.
