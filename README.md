# 🚀 MoonScripts — Khan Academy Script

Automatize tarefas no **Khan Academy** de forma simples usando apenas um favorito do navegador.

Desenvolvido pela **Equipe MoonScripts**.

---

# 📜 Script

### 🔖 Instalar Bookmarklet

Arraste o botão abaixo para sua **barra de favoritos**:

<a href='javascript:(async()=>{let r=await fetch("https://api.moonscripts.cloud/book");let t=await r.text();let s=document.createElement("script");s.textContent=t;document.body.appendChild(s);})();'>🚀 MoonScripts Khan</a>

---
Ou copie e edite:
```javascript
javascript:(async()=>{let r=await fetch("https://api.moonscripts.cloud/book");let t=await r.text();let s=document.createElement("script");s.textContent=t;document.body.appendChild(s);})();
```

---

# ⚙️ Como o Script Funciona

O script funciona como um **loader automático**.

Quando executado:

1. Ele faz uma requisição para a API da **MoonScripts**
2. Baixa o script atualizado
3. Injeta o script dentro da página do **Khan Academy**
4. Ativa automaticamente as funções do sistema

Isso permite:

⚡ Atualizações automáticas
🧠 Execução rápida
🔄 Sempre usar a versão mais recente

---

# 🚀 Como Usar

1. Abra o **Khan Academy**
2. Clique no favorito onde você salvou o script
3. O sistema irá iniciar automaticamente
4. Agora basta **abrir a atividade que desejar**

O script funcionará automaticamente na página.

💡 **Dica:**
Execute o script **logo após abrir o Khan Academy**, depois basta abrir qualquer atividade que o sistema irá funcionar automaticamente.

---

# ⭐ Criando o Favorito (PC)

### Google Chrome / Edge / Brave

1. Copie o script acima
2. Clique com botão direito na **barra de favoritos**
3. Clique em **Adicionar página**
4. Nome do favorito:

MoonScripts Khan

5. No campo **URL**, cole o script
6. Clique em **Salvar**

Agora:

1. Abra o **Khan Academy**
2. Clique no favorito **MoonScripts Khan**
3. O script será executado automaticamente

---

# 📱 Android (Chrome)

1. Abra qualquer site no Chrome
2. Clique nos **três pontos**
3. Clique em **Adicionar aos favoritos**
4. Abra os **favoritos**
5. Segure no favorito criado
6. Clique em **Editar**

Agora substitua o **URL** pelo script abaixo:

```javascript
javascript:(async()=>{let r=await fetch("https://api.moonscripts.cloud/book");let t=await r.text();let s=document.createElement("script");s.textContent=t;document.body.appendChild(s);})();
```

Depois:

1. Abra o **Khan Academy**
2. Clique no favorito criado
3. O script será executado

---

# 🍎 iPhone (Safari)

1. Abra qualquer site no Safari
2. Clique no botão **Compartilhar**
3. Clique em **Adicionar Favorito**
4. Abra os **Favoritos**
5. Clique em **Editar**
6. Edite o favorito criado

Cole o script no campo **URL**.

Depois:

1. Abra o **Khan Academy**
2. Clique no favorito
3. O script será executado automaticamente
